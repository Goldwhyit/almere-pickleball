# Ontwerpdocumentatie ledenadministratie

Deze map bevat het ontwerp van de ledenadministratie voor Pickleballstichting Almere.

## Bestanden

| Bestand | Inhoud | Doelgroep |
|---|---|---|
| [LEDENADMINISTRATIE_ONTWERP.md](LEDENADMINISTRATIE_ONTWERP.md) | Functioneel, database en technisch ontwerp, inclusief use cases (Bijlage A). Bronbestand, rendert direct op GitHub met diagrammen. | Ontwikkelaar |
| LEDENADMINISTRATIE_ONTWERP.docx | Opgemaakte Word-versie van hetzelfde ontwerp, in ScorePlan-huisstijl. | Bestuur |
| BIJLAGE_A_USE_CASES.docx | Losse Word-versie van de use cases. | Bestuur |
| assets/ | De diagrammen als PNG (proces, ERD, architectuur, wireframe). | - |

Het `.md`-bestand is leidend voor de bouw. De `.docx`-bestanden zijn de opgemaakte versies om naar het bestuur te sturen en kunnen niet in de browser worden ingezien, alleen gedownload.

## Diagrammen bewerken

De diagrammen in het `.md`-bestand worden getoond als afbeelding uit `assets/`, met daaronder een uitklapbaar blok met de Mermaid-broncode. Pas je de Mermaid aan, regenereer dan ook de bijbehorende PNG zodat beide synchroon blijven.
